  +--- ADD NEW STUDENT PROFILE -----------------------------------+

  Name
    Allowed : A-Z   a-z   space   -
    Rules   : Minimum two words  |  Each word min 2 chars
              Hyphen ( - ) must be between two letters
              No character repeated more than twice in a row
    Length  : 5 to 29 characters
    > 
  Date of Birth
    Format  : DD/MM/YYYY   e.g. 14/03/2005
    Note    : Student must be 14 to 21 years old
    > 
  ID
    Allowed : A-Z   a-z   0-9   ( no spaces or symbols )
    Length  : 4 to 9 characters
    > 
  Address
    Allowed : A-Z   a-z   0-9   space   .   ,   -   /
    Rules   : No consecutive identical symbols   e.g. // or ..
    Length  : 10 to 39 characters
    > 
  Guardian Phone
    Allowed : 0-9   ( digits only )
    Note    : Use 00 prefix for international   e.g. 0020100123456
    Length  : 6 to 15 digits
    >
